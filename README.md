
# Investment Tracker

A simple application to track and manage your investment portfolio.

## Features

- Add and remove investments
- Track portfolio performance
- View investment summaries
- Monitor returns and gains/losses

## Getting Started

### Prerequisites

- Python 3.8+
- Required dependencies (see `requirements.txt`)

### Installation

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Run the application

## Usage

```bash
python main.py
```
